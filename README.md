# UtilityCraft: Heavy Machinery v0.1.0
---

## ADDITIONS

### Power Condenser Matrix
- Added the **Power Condenser Matrix** multiblock.
- Uses **steel casing**.
- Accepts **energy cells** or any tier of **Power Condensing Unit**.
- Requires **energy ports** to insert or extract energy.

### Thermal Reactor
- Added the **Thermal Reactor** multiblock.
- Requires **lava** (will be changed to fuel in a future update) and **water** as a coolant to operate.
- Interacting with the controller allows toggling between on/off and adjusting the **fuel burn rate**.
- Requires **heat conduits**, **fluid cells**, **energy cells or power condensing units**, **vents**, and a **Thermo Core**.
- Uses **bronze casing**.

## NOTES
- No crafting recipes are included; this version is for creative testing only.
- Requires **UtilityCraft v3.3.0+** to function.
