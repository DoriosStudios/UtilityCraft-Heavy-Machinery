# UtilityCraft: Heavy Machinery  

- Latest Release: https://github.com/DoriosStudios/UtilityCraft-Heavy-Machinery/tag/v0.4.0
---
**UtilityCraft: Heavy Machinery** is an expansion pack for **[UtilityCraft](https://github.com/DoriosStudios/UtilityCraft)**, aiming for a new progression line for end-game using **modular, multiblock** machines with better management and, obviously, more power and capanilities. 

Currently, this project adds:
- A new active generator, called **Thermo Reactor**, that uses a coolant and lava to generate energy  
- Ports of UtilityCraft machines to multiblock machines  
- Two new materials, these being **Bronze** alloy and **Tin** ore.  

And that's not everything! There are more content planned for Heavy Machinery, like
- A radiation system, with Nuclear Reactors and the effects caused by it
- A new armor set, called **Utility Exo**, with extra features and the best protection in the entire game
- More ports and additions.  

And certainly more!
